\subsection{Problem 1: Multi-layer Perceptron (MLP) on ReducedMNIST}

The objective of this problem is to classify the ReducedMNIST digits using MLP classifiers with 1, 3, and 4 hidden layers. The dataset follows the assignment split of 1000 training images per digit and 200 testing images per digit. Each image was represented using the DCT feature vector used in Assignment 1, then passed to fully connected neural networks.

\subsubsection{Methodology}
The three required depths were trained under the same feature extraction and evaluation protocol. The hidden layers used ReLU activations, and the main configurations used Batch Normalization and Dropout for regularization. The Adam optimizer minimized categorical cross-entropy with an initial learning rate of $0.001$. Two extra diagnostic runs were also executed: one without regularization and one with a higher learning rate, to show the effect of these hyperparameters.

\subsubsection{Results and Observations}
All three required MLP depths achieved very similar performance. The 1-hidden-layer model reached the highest measured accuracy at $97.3\%$, while the 3-hidden and 4-hidden models rounded to the same reported accuracy but required more training time. This indicates that the DCT representation is already discriminative for ReducedMNIST, so extra depth mainly increases computation rather than improving accuracy.

The two diagnostic experiments confirm that regularization and learning rate selection matter. Removing Batch Normalization and Dropout reduced the 3-hidden-layer result to $96.7\%$, and increasing the learning rate to $0.01$ reduced it to $96.6\%$. Therefore, the reported MLP setup satisfies the required 1-, 3-, and 4-hidden-layer experiments, with timing reported in milliseconds and accuracy rounded to one decimal place.
