AraVec 3.0 — أكبر نموذج تضمين عربي مفتوح المصدر

مدرب على أكثر من 1,169,075,128 رمز (token) من تغريدات تويتر ومقالات ويكيبيديا العربية. يوفر 16 نموذجاً مختلفاً بأبعاد 100 و300.

1. CBOW — Continuous Bag of Words

يتنبأ CBOW بالكلمة المستهدفة بناءً على السياق المحيط بها. يأخذ نافذة من الكلمات المجاورة ويتعلم تمثيل الكلمة المركزية. أسرع في التدريب وأفضل للكلمات الشائعة.

2. Skip-gram

يتنبأ Skip-gram بالكلمات السياقية بناءً على الكلمة المركزية. يعمل بشكل أفضل مع الكلمات النادرة وعينات التدريب الصغيرة — النموذج المفضل في AraVec 3.0 لدقته العالية في التشابه الدلالي.

3. بنية الشبكة العصبية
4. مقاييس المسافة
تشابه جيب التمام
sim(A,B) = A·B / (|A|·|B|)

المستخدم في AraVec للتشابه الدلالي. القيمة بين -1 و+1.

مسافة إقليدس
d(A,B) = √Σ(aᵢ-bᵢ)²

تقيس المسافة الهندسية المباشرة في الفضاء ثلاثمئة الأبعاد.

خط معالجة النصوص العربية — Arabic NLP Pipeline

1

جمع البيانات الخام

تغريدات تويتر العربية + مقالات ويكيبيديا العربية → أكثر من مليار رمز.

corpus = twitter_ar + wikipedia_ar tokens_total ≈ 1,169,075,128
2

التطبيع — Text Normalization

إزالة التشكيل، توحيد أشكال الألف، إزالة الرموز والأرقام الأجنبية.

إزالة التشكيل: كَتَبَ → كتب توحيد الألف: أإآ → ا إزالة اللاتيني: hello123 → [محذوف]
3

الترميز — Tokenization

تقطيع الجمل إلى كلمات مع دعم الـ n-grams (أحادية ومتعددة) في AraVec 3.0.

"صباح الخير يا مصر" → ["صباح", "الخير", "يا", "مصر"] → unigrams + bigrams: "صباح_الخير"
4

تدريب Word2Vec باستخدام Gensim

نافذة سياقية = 5، أبعاد = 300، الحد الأدنى للتكرار = 5.

from gensim.models import Word2Vec model = Word2Vec( sentences=corpus, vector_size=300, window=5, min_count=5, sg=1, # Skip-gram workers=8 )
5

تخفيض الأبعاد للتصور — t-SNE

تحويل المتجهات ذات الـ 300 بُعد إلى 3 أبعاد قابلة للتصور مع الحفاظ على البنية المحلية.

from sklearn.manifold import TSNE xyz = TSNE( n_components=3, perplexity=30, n_iter=1000 ).fit_transform(vectors_300d)
6

تحميل النموذج واستخدامه

تحميل نموذج AraVec المُدرب مسبقاً وإجراء استعلامات التشابه.

from gensim.models import KeyedVectors model = KeyedVectors.load('aravec_sg_300.mdl') # أقرب الكلمات model.most_similar('ملك', topn=5) # حساب التناظر model.most_similar( positive=['ملك', 'امرأة'], negative=['رجل'] )
